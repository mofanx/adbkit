# ADB Kit 用户手册

> 版本：v1.4.0  
> 适用范围：Android 8.0 (API 26) 及以上

## 快速开始

1. 在目标设备上开启 **开发者选项 > USB 调试** 或 **无线调试**。
2. 打开 ADB Kit，首页输入目标设备的 `IP:端口`（默认 `5555`）或连接 USB。
3. 点击「连接」进入设备管理主界面。
4. 首次启动会弹出引导页，可在 **设置 > 帮助** 中再次查看。

## 主要功能

### 设备连接
- **USB**：直接插线，授权后即可连接。
- **WiFi/无线调试**：输入 `IP:5555` 或通过网络扫描发现设备。
- **历史记录**：首页下拉显示最近连接地址，点击快速连接。

### 设备信息
- 查看电池、存储、CPU、网络、硬件等完整信息。
- 支持一键复制、导出 JSON、系统分享。
- 电池与存储历史趋势以 Canvas 折线图展示。

### 文件管理
- 浏览 `/sdcard`、`/data`、`/system` 等目录（受限目录会提示 root）。
- 支持上传、下载（带进度条）、重命名、复制、移动、删除。
- 支持文本/图片预览，APK 文件可直接安装。
- 长按文件进入多选批量删除。

### 应用管理
- 分类查看用户/系统应用，搜索包名。
- 启动、强制停止、清除数据、卸载、冻结/解冻。
- 批量备份 APK 到 `/sdcard/app_backup_adbkit`。
- 导出应用列表到 `/sdcard/app_list_adbkit.txt`。
- 查看应用图标、版本、权限、组件数量等详情。

### 进程管理
- 实时显示运行进程，每 5 秒自动刷新。
- 按内存、CPU、PID、进程名排序。
- 结束进程后自动刷新。
- 点击详情查看命令行、线程数、CPU 时间等。

### 终端
- 支持 ADB Shell 和 ADB 命令两种模式。
- 命令历史、收藏夹（最多 20 条）持久化。
- 可将收藏夹保存为**命名宏**，一键运行或删除。
- 输出支持搜索、复制、保存、分享，执行中的命令可取消。

### 远程控制
- 基于 scrcpy 协议的 H.264 投屏。
- 支持触控、手势、虚拟按键、键盘输入、剪贴板粘贴。
- 可调整分辨率、码率、帧率；卡顿时可一键启用弱网预设。
- 支持熄屏控制，多种 fallback 保证兼容性。

### Fastboot
- 镜像文件选择器，支持 MD5 校验与期望 MD5 比对。
- 分区备份/还原（需 root 或对应权限）。
- 刷入、擦除、重启、解锁/锁定 BL。
- 输出可保存到本地。

### 实用工具
- 截图、录屏（支持中途停止）。
- 亮度、屏幕密度/分辨率、导航栏/状态栏控制。
- 打开链接、启动应用、当前 Activity、Logcat、系统属性（getprop/setprop）。
- WiFi/蓝牙/飞行模式开关（带状态回显）。

## 设置

- **ADB/Fastboot 路径**：自动检测或手动配置。
- **默认端口**：无线连接端口，默认 `5555`。
- **自动连接**：启动时尝试连接上次设备。
- **深色模式 / 动态颜色**：跟随系统或手动切换。
- **界面语言**：跟随系统、中文、英文。
- **危险操作确认**：卸载、擦除、重启等二次确认弹窗。
- **保存历史**：命令历史与连接历史的持久化开关。

## 常见问题

**Q：无法连接设备？**
- 确认目标设备已开启 USB/无线调试并授权此主机。
- 检查 IP 与端口，确保两台设备在同一局域网。
- 撤销「USB 调试授权」后重新授权。

**Q：文件/应用管理操作失败？**
- 某些系统目录需要 root 权限。
- 应用安装/卸载等操作可能需要设备侧权限或处于正常启动状态。

**Q：远程控制画面卡顿？**
- 降低分辨率、码率或帧率。
- 开启弱网模式预设。
- 确保两台设备网络稳定。

**Q：如何反馈问题？**
- 进入 **设置 > 反馈** 发送邮件，或查看崩溃日志并分享。

## 安全提示

- Fastboot 刷机、分区操作存在风险，请仔细核对分区与镜像。
- 危险操作均提供二次确认，可在设置中关闭，但不建议。
- 不要在未授权的设备上执行破坏性命令。
